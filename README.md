# Dart OOP Guide 

This repository is a practical guide to **Object-Oriented Programming (OOP) in Dart**.  
It contains clear explanations and hands-on examples covering the core `OOP` concepts, written in a simple and beginner-friendly way.

The goal of this repo is to build a **strong OOP foundation** before moving to **Flutter development**.
